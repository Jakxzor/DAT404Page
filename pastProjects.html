Coffee Game

This was a project about brainstorming design ideas. We were given a topic and told to design layouts for how the game might look. I chose a coffee based game that would work along the lines of "Cooking Mama", where the user is given a type of coffee to make and they use buttons on screen to make that coffee, getting points for speed and accuracy.
